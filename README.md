# 🧠 Nueradash  
### *The New Era of AI Dashboards*

**Nueradash** is a full-stack, AI-powered analytics platform that helps businesses visualize, analyze, and predict their performance — all in one place.  
Upload your sales data and let Nueradash do the heavy lifting: from generating clean dashboards to forecasting future trends using machine learning.

---

## 🚀 Features

- 📊 **Interactive Dashboard** — View key metrics, charts, and insights in real-time.  
- 🤖 **AI Forecasting** — Predict sales trends using built-in ML models.  
- 📈 **Insight Generation** — Get plain-English summaries of performance data.  
- ☁️ **Full-Stack Architecture** — React (frontend), Node.js (backend), MongoDB (database), FastAPI (ML microservice).  
- 🔐 **User Authentication** — Secure login and data upload system.  
- 🎨 **Modern UI/UX** — Futuristic glassmorphism design with Tailwind CSS and Chart.js.

---

| Layer | Technology |
